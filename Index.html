HTML Basics

HTML stands for Hyper Text Markup Language. It is the standard language used to create web pages. HTML is often used in conjunction with CSS for styling and JAVASCRPIPT for interactivity in web paegs. It provides the structure of a webpage using elements like headings, paragraphs, lists, and links. 

                    <h1>Hello World</h1>
                        <p>This is a paragraph</p>
                        <title>My Pgae</title>

CSS styling

CSS stands for Cascading Style Sheet. It add style-colors layouts, spacing to your HTML content. CSS allows you to separate presentation from structure, making it easier to mintain and update the dessign of a media webpage. some common CSS concept include: selectors:Target specific HTML elements to apply styels. properties:Defines the styles, such as color font-size, or margin. Values:Specify the value for a property, such as color code or font-family. 

                        
                            body {font-family: sans-serif; color: #333}
                            image {image-size: 100%; background color: white;
                                 padding: 20px; width: 300px;}
